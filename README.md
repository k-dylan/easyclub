EasyClub
-----
基于 koa2 + mongodb 的轻量级论坛系统

INSTALL
----
1. 安装`mongodb`,`Redis`和`node`(>=v7.1.0)
2. clone 该仓库到本地
3. 复制 `config.default.js` 为 `config.js` 中的配置选项
4. 运行 `npm install` 安装依赖包
5. 运行 `npm run test` 跑测试
6. 执行`npm start` 使用 `nodemon` 启动，执行 `npm pm2` 使用 `pm2` 启动
